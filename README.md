# Projects

This repo is a **vibe coding** experiment: ideas sketched quickly, structure evolving as we go, and “good enough to learn” prioritized over polished production rigor. Expect rough edges, changing directions, and notes-to-self instead of a formal spec.

## What’s here

- **Nexus** — Next.js app (Sentry webhooks, automation direction, Postgres / LangGraph pieces). See [`Nexus/IMPLEMENTATION.md`](Nexus/IMPLEMENTATION.md).
- **sentry-playground** — informal scratch / experiments around Sentry.

## Philosophy

- Ship small slices, refactor when it hurts.
- Docs and env examples might lag the code; that’s part of the vibe.
- Not a template repo — more a personal lab.

## If you’re not me

Clone at your own risk; there is no promise of stability, tests, or support.
